Travel Website – Responsive Travel Landing Page

## About

Travel Website is a responsive travel landing page designed to showcase destinations, travel experiences, and modern UI layout patterns.

This project focuses on layout structuring, responsive design, hero sections, destination presentation, and clean UI styling using pure HTML and CSS.

It was built as a frontend UI project to practice real-world landing page layouts commonly used in travel and tourism websites.

The project emphasizes clean structure, visual presentation, and responsive behavior across desktop and mobile devices.

This project will continue to evolve with UI improvements and additional sections as part of frontend development practice.

## Project Structure / Components

Navbar: Responsive navigation bar

Hero Section: Main landing banner with call-to-action

Destinations Section: Featured travel destinations

Content Sections: Travel highlights and layout blocks

Footer: Website footer with navigation links

Responsive CSS: Media queries for mobile and tablet layouts
